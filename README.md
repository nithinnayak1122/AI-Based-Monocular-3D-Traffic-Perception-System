# AI-Based Monocular 3D Traffic Perception System

This project implements a real-time monocular 3D traffic perception system using YOLOv8 and OpenCV. The system detects and tracks vehicles from a single camera feed, estimates depth, and generates traffic analytics for intelligent transportation and smart city applications.

## Features
- Real-time vehicle detection using YOLOv8
- Vehicle tracking and counting
- Monocular depth estimation
- Distance and traffic analysis
- Dashboard visualization
- Smart city traffic monitoring

## Tech Stack
- Python
- YOLOv8
- OpenCV
- NumPy
- Pandas
- Matplotlib
